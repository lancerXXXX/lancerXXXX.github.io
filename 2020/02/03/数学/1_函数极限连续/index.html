1. 函数

1.1. 关于奇偶性

任一定义在对称于原点的数集X上的函数 f(x), 必可以分解成一奇一偶函数之和

  • 求导一次奇偶性改变一次

  • 常见奇函数 $\ln\frac{1-x}{1+x}, \ln(x+\sqrt{1+x^2}), \frac{e^x- 1}{e^x+1}$

  • $f(x)是奇函数\rightarrow f’(x) 是是偶函数; $

  • $f(x)是偶函\leftrightarrow f’(x) 是奇函数; $

  • 解释: 连续的奇函数其原函数 都是 偶函数;
    连续函数一定有原函数: 如果f(x)连续,
    则 $(\int_{a}^{x}f(x)dx)’=f(x), 所以变上限积分至少是一个原函数$

  • $f(x)连续, 若是奇函数, 则 \int_{a}^{x} f(t) dt是偶函数$
  • $F(x)=\int{0}^{- x}f(t)dt\overset{\text{t=- u}}{=}\int{0}^{- x} [- f(u)(- du)] =\int_{0}^{x}f(u)du$

  • $f(x)连续, 若是偶函数, 则\int_{0}^{x}f(t) dt是奇函数$

  • 解释: 连续的偶函数其原函数 之一 是奇函数;

2. 极限

2.1. 无穷小的比较

$设x \rightarrow 时\alpha(x)和\beta(x)为两个无穷小, \beta(x)\neq 0, \alpha(x)不恒等于0, 设\lim{x\rightarrow } \frac{\alpha(x)}{\beta(x)}=A, $

1.$若A \neq 0, 则称x\rightarrow 时, \alpha与\beta为同阶无穷小.$
2.$若A =0 , 则称x\rightarrow 时, \alpha与\beta为等价无穷小, 记成x\rightarrow 时, \alpha(x)\sim \beta(x).$
3.$若A=0, 则称x\rightarrow 时m, \alpha(x)为\beta的高阶无穷小, 记成x\rightarrow 时, \alpha(x)=o(\beta(x)) .$
4.$如果\lim_{x\rightarrow } \frac{\alpha(x)}{\beta(x)}=\infty , 则称x\rightarrow *时, \alpha(x)为\beta(x)的低阶无穷小.$

  1. 保号性推论
  • $设存在的一个去心邻域, 在此邻域内f(x)\geqslant 0( 或\leqslant 0), 且\lim_{x\rightarrow } f(x) 存在且等于A, 则A \geqslant 0( \leqslant 0 ).$
  • $f(x)和g(x)在某去心邻域内连续, \lim{x\rightarrow x_0} f(x) =a, \lim{x\rightarrow x_0} f(x)=b$
  • $a-b>0\rightarrow 去心邻域内f(x)-g(x)>0$
  • $去心邻域内f(x) -g(x)>0\rightarrow a-b\geqslant 0$
